# README Positioning Notes

This short note records the README positioning used for the portfolio front page.

## Positioning

Local AI Workspace is presented as a local-first AI workspace, not only as a chatbot.

The README highlights that the project combines:

- chat UI
- persistent memory
- document/source retrieval
- response planning
- answer grounding
- safety checks
- audit logs
- eval and trace views
- health checks
- backup/export workflows

## Boundary

The README must stay realistic:

- portfolio-stage project
- not production-ready without additional hardening
- no enterprise-readiness claim
- no external certification claim

## Goal

The public front page should be slightly more product-like while still being honest about scope and maturity.
